# 太阳能草坪灯电源系统设计——从光伏充电到 LED 驱动

> 电源系统是太阳能草坪灯的"心脏"，承担着从太阳能收集、电池管理到负载供电的全链路能量调度。本文详细阐述光伏恒流恒压充电、2S 锂电池保护、LDO 双路稳压以及 LED 恒流驱动四大模块的设计思路与参数计算。

---

## 1. 电源系统总体架构

太阳能草坪灯的电源系统需要解决一个核心矛盾：**输入端（太阳能板）的输出随光照剧烈波动，而输出端（LED 负载与逻辑电路）要求稳定供电**。为此，本文采用"充电管理 → 电池缓冲 → 分域稳压 → 负载驱动"的四层架构：

```
太阳能板 → 防反冲 → Buck-Boost CC/CV充电 → 2S锂电(7.4~8.4V)
                                                    ↓
                                              ┌─────┼─────┐
                                              ↓     ↓     ↓
                                            +5V   +3V   LED驱动
                                           LDO①  LDO②  PT4115
                                              ↓     ↓     ↓
                                            DVCC  AVCC  LED负载
                                           (逻辑) (模拟) (照明)
```

### 1.1 电压域划分

| 电压域 | 电压范围 | 用途 | 噪声敏感度 |
|:---:|:---:|:---|:---:|
| V_BAT | 7.4–8.4V | 电池直出，LED 驱动输入 | 低 |
| +5V (DVCC) | 5.0V ±2% | 数字逻辑、定时器供电 | 中 |
| +3V (AVCC) | 3.0V ±2% | 光控采样、比较器阈值 | 高 |

### 1.2 AVCC 与 DVCC 分离的必要性

模拟信号对电源纹波和地线噪声极为敏感。若 AVCC 与 DVCC 共用供电路径，数字电路翻转时产生的瞬态电流会通过公共阻抗注入模拟电源，引起：

- 比较器参考电平波动 → 阈值漂移 → 误触发；
- 光敏电阻采样噪声 → 光控判断失准。

因此，本文在 +5V 和 +3V 输出侧分别串联磁珠，与去耦电容构成 π 型滤波结构，实现 AVCC/DVCC 的噪声隔离。磁珠选型需关注：

- **直流电阻（DCR）**：应尽量小，减少压降；
- **额定电流**：大于支路最大工作电流；
- **阻抗特性**：DVCC 侧选小阻抗磁珠（抑制数字开关噪声），AVCC 侧选大阻抗磁珠（提高模拟供电纯净度）。

## 2. 光伏恒流恒压充电模块

### 2.1 为什么不用 MPPT？

MPPT（最大功率点跟踪）技术能使光伏面板始终工作在 U-I 特性曲线的峰值功率点，但实现 MPPT 需要已知所用光伏设备的 U-I 特性曲线。本文所用太阳能板仅掌握最大开路电压和短路电流参数，无法设计实现 MPPT 功能。

因此，本文退而求其次，采用**恒流恒压（CC/CV）充电**方案，配合锂电池的充电特性需求。

### 2.2 SC8701 Buck-Boost 充电 IC

本文选用南芯半导体的 **SC8701QDER** 作为充电管理 IC。该芯片具有以下优势：

- 支持 Buck-Boost 拓扑，输入电压可高于或低于输出电压；
- 通过 PWM 与阻容网络控制和限制输入/输出电流与电压；
- QFN-32 封装，便于 SMT 生产；
- 具有成熟的典型应用电路。

#### 2.2.1 工作原理

输入功率通过 VIN 接入，经电容滤波后通过输入电流采样电阻（SNS1P/SNS1N）与 SC8701 相连。功率在 SC8701 调制下受控通断，完成 Buck-Boost 变换。输出侧同样设置电流采样电阻（SNS2P/SNS2N）提供反馈，最终经输出滤波电容输出恒压恒流功率。

#### 2.2.2 关键参数设计

**开关频率**：通过 FREQ 引脚对地电阻设定：

| 电阻阻值 | 开关频率 |
|:---:|:---:|
| 0Ω | 200 kHz |
| 68kΩ | 400 kHz |
| 开路 | 600 kHz |

本文选取 $R_{SFREQ} = 68\text{k}\Omega$，开关频率为 400 kHz，兼顾效率与纹波。

**电流采样与限流**：采样电阻与限流电阻需满足比例关系：

$$\frac{R_{SNS}}{R_{ILIM}} = \frac{R_{sense}}{R_{sense\_pin}}$$

其中 $R_{sense}$ 推荐在 5–10 mΩ 范围内。本文取 $R_{sense} = 10\text{m}\Omega$，$R_{sense\_pin} = 100\Omega$。

SC8701 的电流限制遵循：

$$I_{LIM} = \frac{V_{REF}}{R_{ILIM}}$$

其中 $V_{REF} = 1.21\text{V}$ 为内部参考电压。需要注意的是，该公式并非简单的欧姆定律——SC8701 内部集成了电流采样放大器，采样电阻上的微弱压降经内部增益放大后与 $V_{REF}$ 比较，因此 $R_{ILIM}$ 设定的是内部比较阈值而非直接流过的电流。

- **输入电流限制**：不做限制，$R_{ILIM1} = 0\Omega$；
- **输出电流限制**：采用 68–110 kΩ 可变电阻 $R_{a5}$ 与定值电阻 $R_{ILIM2}$ 组合，输出电流可在 1.21–1.78 A 之间调节，适配不同容量的锂电池组。

**死区时间**：结合功率 MOSFET（HSBB6066）特性，选取死区时间 40 ns，对应 $R_{DT} = 68\text{k}\Omega$。

**功率电感**：选取 YSPI1040T-3R3M，$L = 3.3\mu\text{H}$，满足 SC8701 数据手册要求并兼顾 RoHS。

**输入输出滤波**：采用 220μF 铝电解电容并联两个 22μF 与一个 100nF 的 MLCC，额定电压均高于 16V。

### 2.3 防反冲保护

在太阳能板输入端串联肖特基二极管 **SS34**（3A/40V），防止夜间电池通过太阳能板反向放电。SS34 具有较低的正向压降（约 0.5V）和快速的反向恢复时间，在确保电源效率的前提下提供可靠保护。

## 3. 2S 锂电池保护模块

### 3.1 保护需求

2S 锂电池组（两节串联）对充放电控制要求严格，必须防止：

- **过充**：单节电压 > 4.25V，有起火风险；
- **过放**：单节电压 < 2.5V，永久损伤电池；
- **过流/短路**：电流超过安全阈值。

对于长期无人值守的户外灯具，保护电路直接关系到系统安全和使用寿命。

### 3.2 HY2120 保护 IC

本文选用 **HY2120** 系列电池保护专用 IC，该芯片专为双节锂离子/锂聚合物电池设计，集成了：

- 高精度电压检测器；
- 延迟电路；
- 静电保护电路（ESD）。

可同时兼顾电池保护与局部 EMC 需求。开关 MOSFET 选用与前级相同型号的 HSBB6066，保证 BOM 简化与生产便捷。

## 4. LDO 双路降压稳压模块

### 4.1 LDO 选型

低压差线性稳压器（LDO）具有结构简单、噪声低的优势，适合为对电源质量要求较高的逻辑和模拟电路供电。

| 输出电压 | IC 型号 | 封装 |
|:---:|:---|:---:|
| +5V | HR1117-5.0 | SOT-89 |
| +3V | AMS1117-ADJ | SOT-89 |

两者封装与引脚配置完全一致，仅稳压方式不同：HR1117-5.0 为固定输出版，AMS1117-ADJ 为可调输出版。

> **关于型号说明**：论文原文将 +3V 侧 LDO 标注为"AMS1117-3.3"（固定 3.3V 输出版），但实际电路采用可调输出版并通过外接分压电阻实现 3.0V 输出。因此本文使用 AMS1117-ADJ 以准确反映实际设计。

### 4.2 输出电压计算

AMS1117-ADJ 的输出电压由外接分压电阻决定：

$$V_{OUT} = V_{REF} \cdot \left(1 + \frac{R_2}{R_1}\right) + I_{ADJ} \cdot R_2$$

其中 $V_{REF} = 1.25\text{V}$，$I_{ADJ}$ 为调整端电流。当 $I_{ADJ}$ 足够小时：

$$V_{OUT} \approx 1.25 \times \left(1 + \frac{R_2}{R_1}\right)$$

为取得 +3V 输出，取 $R_1 = 100\Omega$，$R_2 = 140\Omega$：

$$V_{OUT} = 1.25 \times \left(1 + \frac{140}{100}\right) = 1.25 \times 2.4 = 3.0\text{V}$$

同时需满足数据手册限制：

$$\frac{V_{OUT}}{R_1 + R_2} \geq 5\text{mA}$$

验证：$\frac{3.0}{100 + 140} = 12.5\text{mA} \geq 5\text{mA}$ ✓

### 4.3 滤波与隔离

在 +5V 与 +3V 输出侧分别串联磁珠，磁珠远离 LDO 一侧并联 1μF 接地去耦电容，构成 π 型滤波结构。在 PCB Layout 中，AVCC 与 DVCC 之间分割 VCC 与 GND 内电层，实现电源域的物理隔离。

## 5. LED 恒流驱动模块

### 5.1 为什么需要恒流驱动？

LED 的亮度由正向电流决定。若采用简单的电阻限流 + MOSFET 开关方案：

- 2S 电池电压在 7.4–8.4V 之间变化，导致 LED 电流大幅波动；
- 亮度不稳定，效率低，发热大。

因此，本文采用专用 LED 驱动 IC 实现 Buck 恒流驱动。

### 5.2 PT4115 驱动 IC

本文选用 **PT4115**（UMW 出品），具有：

- 最高 30V / 1.2A 输出能力；
- 过温保护；
- PWM 调光功能（本文仅用作开关使能）。

### 5.3 输出电流设计

本文选取 9 个 LED 并联作为负载，每个 LED 正向压降 2.7V、正向电流 20mA，总驱动电流约 180mA。

PT4115 的输出电流由采样电阻 $R_s$ 决定：

$$I_{OUT} = \frac{0.1}{R_s}$$

取 $R_s = 560\text{m}\Omega$：

$$I_{OUT} = \frac{0.1}{0.56} \approx 178.57\text{mA}$$

略低于 180mA，属于相对欠流设计，有利于延长 LED 寿命。

### 5.4 功率电感选取

由数据手册知，当 $I_{OUT} < 0.4\text{A}$ 时，功率电感选取 68–100μH。本文选取 $L = 100\mu\text{H}$。

续流二极管选用 SS34 肖特基二极管，满足快速恢复和正向导通电流要求。

### 5.5 光耦隔离驱动

逻辑电路与负载电路之间通过 **PC357C** 全隔离光耦实现信号传递：

- 逻辑侧：1kΩ 输入保护电阻；
- 负载侧：100Ω 上拉电阻 + 10kΩ 下拉电阻 + 22Ω LED 分压电阻。

该设计完全避免了两侧电源和负载的耦合，提高了 EMC 可靠性。

## 8. EMC 设计要点

电源系统是 EMC 设计的重点区域，主要干扰源包括：

1. **Buck-Boost 开关节点**：高 $dv/dt$ 和 $di/dt$，产生尖峰与纹波；
2. **LED 驱动电路**：负载变化时的电流波动；
3. **光控/时控模拟信号**：对噪声敏感，易受大电流回路耦合干扰。

### 8.1 抑制措施

| 措施 | 目标 | 实施方式 |
|:---|:---|:---|
| 电源去耦 | 抑制芯片电源噪声 | 每个芯片 VCC 引脚附近放置 0.1μF MLCC + 1–10μF 旁路电容 |
| 地参考面分区 | 减少回流串扰 | PGND / AGND / DGND 分割，0Ω 电阻星形互联 |
| 磁珠滤波 | 抑制高频噪声 | AVCC/DVCC 支路串联磁珠，与去耦电容构成 π 型滤波 |
| 功率回路紧凑 | 降低辐射干扰 | 电感、MOSFET、二极管所在回路面积最小化 |
| 信号隔离 | 避免耦合 | 模拟检测节点远离开关节点，时钟信号线与数字信号线垂直交叉 |

### 8.2 地平面分割策略

在四层板设计中，内电层分为电源参考层和地参考层。地参考层进一步分割为：

- **PGND**（保护地）：功率回路回流路径；
- **AGND**（模拟地）：光控采样与比较器回流路径；
- **DGND**（数字地）：逻辑电路回流路径。

三者在顶层通过 0Ω 连接电阻实现星形单点互联，最大程度避免各类信号或功率回流时的串扰。

## 9. 小结

本文完成了太阳能草坪灯电源系统的完整设计，涵盖光伏 CC/CV 充电（SC8701）、2S 锂电池保护（HY2120）、双路 LDO 稳压（HR1117/AMS1117-ADJ）和 LED 恒流驱动（PT4115）四大模块。通过 AVCC/DVCC 分离、磁珠滤波、地平面分割等 EMC 措施，确保了系统在户外复杂电磁环境下的稳定运行。所有元器件选型均优先考虑 RoHS 合规性，兼顾了环保要求与工程可实现性。
