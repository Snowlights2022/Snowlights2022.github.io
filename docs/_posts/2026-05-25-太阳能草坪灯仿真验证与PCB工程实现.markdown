# 太阳能草坪灯仿真验证与 PCB 工程实现

> 从原理图到实物，仿真验证是确保设计正确性的关键环节，而 PCB 工程实现则决定了设计能否可靠落地。本文围绕 Multisim 仿真验证、立创 EDA 设计流程、四层板 Layout 策略、EMC 工程实践以及系统测试与经济性分析，完整呈现从"纸面设计"到"可制造产品"的工程化路径。

---

## 1. 仿真验证策略

### 1.1 仿真工具选择

本文选用 **NI Multisim** 作为仿真平台，原因如下：

- 内置丰富的模拟/数字器件模型，支持混合信号仿真；
- 支持虚拟仪器（示波器、逻辑分析仪等），便于观测信号波形；
- 与 Ultiboard 联动，可进行 PCB 前仿真验证。

### 1.2 仿真验证层次

仿真验证分为三个层次，由底向上逐级确认：

| 层次 | 验证内容 | 通过标准 |
|:---:|:---|:---|
| 模块级 | 各功能模块独立工作正确性 | 输入输出关系符合设计预期 |
| 接口级 | 模块间信号传递与电平匹配 | 信号完整，无电平冲突 |
| 系统级 | 全链路联合工作正确性 | 状态机转换正确，定时准确，负载受控 |

## 2. 关键模块仿真结果

### 2.1 光控检测模块

**测试方法**：用可变电阻模拟光敏电阻阻值变化，按"白天 → 黄昏 → 夜间 → 黎明 → 白天"的时序调节，观察比较器输出。

**关键观测点**：

- 比较器 A 输出（日落检测）：当模拟照度下降穿越开启阈值时，输出由低变高；
- 比较器 B 输出（日出检测）：当模拟照度上升穿越关断阈值时，输出由低变高；
- 迟滞窗口：两次翻转之间的照度差值。

**仿真结论**：

- 迟滞反馈有效抑制了临界照度下的输出抖动；
- 日落与日出事件被正确识别，输出干净的电平跳变；
- 迟滞窗口宽度与理论计算值吻合，约 5mV。

### 2.2 边沿识别模块

**测试方法**：在比较器输出端施加缓慢变化的电平信号，观察 D 触发器与与非门的输出。

**仿真结论**：

- 下降沿检测电路在输入信号下降沿处产生约 100ns 宽度的负脉冲；
- 脉冲宽度由 D 触发器传播延迟决定，足以触发 ICM7555 定时器；
- 上升沿不产生误触发脉冲。

### 2.3 定时器模块

**测试方法**：向 ICM7555 定时器 TRIG 引脚施加负脉冲，测量 OUT 引脚保持高电平的持续时间。

**定时公式验证**：

$$t_w = \ln 3 \cdot R_T \cdot C_T \approx 1.1 \cdot R_T \cdot C_T$$

| $R_T$ | $C_T$ | 理论值 | 仿真值 | 误差 |
|:---:|:---:|:---:|:---:|:---:|
| 10kΩ | 940μF | 10.34s | 10.21s | 1.26% |
| 50kΩ | 940μF | 51.7s | 50.9s | 1.55% |
| 100kΩ | 940μF | 103.4s | 101.8s | 1.55% |

仿真值与理论值偏差在 2% 以内，验证了定时公式的正确性。

### 2.4 锁定机制验证

**测试方法**：在定时器计时结束后，再次施加触发脉冲，观察系统是否响应。

**仿真结论**：

- 定时结束后，D 触发器锁定 ICM7555 的 RST 引脚为低电平；
- 后续触发脉冲无法重新启动定时器；
- 只有日出复位信号能解除锁定，系统回到待机状态。

### 2.5 系统级联合仿真

将所有模块连接后进行全链路仿真，模拟完整的昼夜循环：

1. **白天**：光敏电阻低阻值，比较器 A 输出低，系统处于待机状态 S0；
2. **日落**：光敏电阻阻值上升，比较器 A 翻转，边沿识别产生触发脉冲，ICM7555 启动计时，LED 点亮，系统进入工作状态 S1；
3. **定时结束**：ICM7555 输出翻转，D 触发器锁定，LED 关闭，系统进入锁定状态 S2；
4. **日出**：光敏电阻阻值下降，比较器 B 翻转，产生复位脉冲，锁定解除，系统回到待机状态 S0。

> 全链路仿真确认了状态机三个状态之间的转换逻辑完全正确，无死锁、无竞争冒险。

## 3. 立创 EDA 设计流程

### 3.1 为什么选择立创 EDA？

本文选用**立创 EDA**（EasyEDA）作为 PCB 设计工具，原因如下：

- **国产化**：完全自主知识产权，符合国产替代趋势；
- **一站式**：原理图 → PCB → BOM → 采购 → 制板，全流程在线完成；
- **元件库丰富**：内置海量国产器件封装与 3D 模型；
- **免费**：标准版完全免费，适合学生与个人开发者。

### 3.2 设计流程

```
原理图设计 → 电气规则检查(ERC) → 网表导入 → PCB布局 → 布线
    → 设计规则检查(DRC) → 铺铜 → 丝印调整 → Gerber输出 → 制板
```

### 3.3 三板分离设计

本文将系统拆分为三块独立 PCB，通过排针/排母连接器互联：

| PCB | 功能 | 尺寸（mm） | 层数 |
|:---|:---|:---:|:---:|
| ChargerBoard | 光伏充电 + 电池保护 | 70 × 50 | 4 |
| LogicBoard | 光控检测 + 边沿识别 + 定时 | 80 × 60 | 4 |
| LoadBoard | LED 恒流驱动 + 负载接口 | 60 × 40 | 4 |

三板分离的优势：

- **功能解耦**：各板独立调试，故障定位方便；
- **EMC 优化**：功率回路与逻辑回路物理隔离；
- **维护便捷**：损坏时仅需更换对应单板；
- **扩展灵活**：可独立升级某一功能模块。

## 4. 四层板 Layout 设计

### 4.1 层叠结构

本文采用四层板设计，层叠结构为：

| 层 | 用途 | 说明 |
|:---:|:---|:---|
| Top | 信号层 + 元件面 | 放置所有 SMD 器件，走信号线 |
| Inner1 | 电源参考层 | 分割为 VCC / AVCC / DVCC 区域 |
| Inner2 | 地参考层 | 分割为 PGND / AGND / DGND 区域 |
| Bottom | 信号层 + 辅助布线 | 补充走线、放置少量器件 |

### 4.2 电源层分割

电源参考层（Inner1）按电压域分割：

```
┌──────────────────────────────────┐
│           VCC (8.4V)             │
│    ┌────────────────────────┐    │
│    │      +5V (DVCC)        │    │
│    │   ┌──────────────┐     │    │
│    │   │  +3V (AVCC)  │     │    │
│    │   └──────────────┘     │    │
│    └────────────────────────┘    │
└──────────────────────────────────┘
```

分割线宽度不小于 20mil，避免不同电压域之间的电场耦合。

### 4.3 地层分割与互联

地参考层（Inner2）按功能域分割为 PGND、AGND、DGND 三个区域，在顶层通过 0Ω 电阻实现星形单点互联：

$$Z_{interconnect} = R_{0\Omega} + j\omega L_{via}$$

0Ω 电阻的直流电阻约 20–50mΩ，在低频段提供近似等电位连接，在高频段其寄生电感与磁珠配合形成滤波网络，有效抑制地回流串扰。

### 4.4 功率走线设计

大电流路径（充电回路、LED 驱动回路）的走线需满足载流能力要求。根据 IPC-2221 标准，1oz 铜厚下外层走线的载流能力为：

$$I_{max} = k \cdot \Delta T^{0.44} \cdot A^{0.725}$$

其中 $k$ 为常数（外层取 0.048），$\Delta T$ 为允许温升（°C），$A$ 为截面积（mil²）。

本文功率走线设计参数：

| 参数 | 数值 |
|:---|:---|
| 铜厚 | 1oz (35μm) |
| 线宽 | 14mil (0.356mm) |
| 允许温升 | 10°C |
| 载流能力 | 约 5A |

### 4.5 过孔设计

过孔在 PCB 设计中承担电气连接与散热双重功能：

- **信号过孔**：孔径 0.3mm / 盘径 0.6mm，用于层间信号连接；
- **散热过孔**：孔径 0.4mm / 盘径 0.8mm，密集排列在功率器件焊盘附近，将热量从顶层导入内层和底层铜面。

散热过孔的热阻近似为：

$$R_{th,via} = \frac{l}{k_{Cu} \cdot A_{Cu}}$$

其中 $l$ 为过孔长度，$k_{Cu} = 385\text{W/(m·K)}$ 为铜的热导率，$A_{Cu}$ 为过孔铜壁截面积。对于 1.6mm 板厚、0.4mm 孔径、1oz 镀铜的过孔：

$$A_{Cu} = \pi \cdot d \cdot t = \pi \times 0.4\text{mm} \times 35\mu\text{m} \approx 0.044\text{mm}^2$$

$$R_{th,via} = \frac{1.6 \times 10^{-3}}{385 \times 0.044 \times 10^{-6}} \approx 94.4\text{°C/W}$$

单个过孔热阻较高，但多个过孔并联可显著降低总热阻。本文在 SC8701 和 PT4115 下方各布置 4–6 个散热过孔，等效热阻降至 15–24°C/W。

## 5. EMC 工程实践

### 5.1 EMI 抑制

电磁干扰（EMI）的主要来源是 Buck-Boost 和 Buck 变换器的开关节点。抑制措施包括：

1. **减小回路面积**：输入电容、开关管、续流二极管、电感构成的回路面积最小化；
2. **RC 缓冲电路**：在开关节点与地之间并联 RC 吸收电路，抑制电压尖峰；
3. **布局紧凑**：高频器件靠近放置，缩短高频走线长度；
4. **铺铜屏蔽**：在开关节点下方内层铺设完整地铜面，提供屏蔽。

### 5.2 EMS 防护

电磁敏感度（EMS）防护确保系统在外部干扰下正常工作：

| 干扰类型 | 防护措施 |
|:---|:---|
| 静电放电（ESD） | 接口处放置 TVS 二极管，HY2120 内置 ESD 保护 |
| 电快速瞬变脉冲群（EFT） | 电源入口处增加共模电感与 X/Y 安规电容 |
| 浪涌（Surge） | 输入端压敏电阻（MOV）钳位 |
| 传导/辐射骚扰 | 磁珠滤波 + 地平面屏蔽 |

### 5.3 信号完整性

对于逻辑信号，需关注：

- **反射**：当走线长度超过信号上升时间的 1/6 时需考虑阻抗匹配。本文逻辑信号频率较低（<1MHz），走线长度远小于临界长度，无需端接；
- **串扰**：相邻信号线间距 ≥ 3 倍线宽，关键信号线周围铺设地线保护；
- **地弹**：每个 IC 的 VCC 引脚附近放置 0.1μF 去耦电容，缩短回流路径。

## 6. 制造与 RoHS 合规

### 6.1 制造参数

| 参数 | 数值 |
|:---|:---|
| 板厚 | 1.6mm |
| 铜厚 | 1oz (35μm) |
| 最小线宽/线距 | 6/6mil |
| 最小过孔 | 0.3mm 孔径 / 0.6mm 盘径 |
| 阻焊 | 绿色 |
| 丝印 | 白色 |
| 表面处理 | 无铅喷锡（HASL Lead-Free） |

### 6.2 RoHS 合规分析

RoHS 2.0 指令限制铅（Pb）、汞（Hg）、镉（Cd）、六价铬（Cr⁶⁺）等有害物质的使用。本文设计中的 RoHS 合规措施：

- **无铅焊接**：采用 SAC305 无铅焊料（Sn 96.5% / Ag 3.0% / Cu 0.5%）；
- **无铅表面处理**：选用无铅喷锡（HASL Lead-Free）替代传统有铅喷锡；
- **元器件选型**：所有元器件优先选择 RoHS 认证型号，如 YSPI1040T-3R3M 电感、SC8701QDER 等；
- **标识**：PCB 丝印标注 RoHS 合规标识。
